# 2AStuff
A collection of remix works that I have assembled in one place. 

Hello, and welcome to my 2A folder. I have begun modofying some of the 3d files that I have found around the ole interwebs and decided it was time for me to place everything in one location where i can reach it at any time. Most of (currently all) of these works are based upon the designs of others. Please pay attention to what folder each design is in, as it will dictate how each file needs to be finished.
